# Titanic Survival Predictor

Predicts whether a Titanic passenger survived based on 
demographic and ticket data.

## Results
- Accuracy: 82.7%
- Algorithm: Random Forest (100 estimators)
- Dataset: 891 passengers · Titanic (Kaggle/public)

## Key Findings
- Female passengers had significantly higher survival rates
- Passenger class strongly correlated with survival
- Age and fare were also meaningful predictors

## Tech Stack
Python · scikit-learn · Pandas · Seaborn · Matplotlib

## Custom Prediction Example
22-year-old male, 3rd class → Did not survive
(Historically consistent result)
